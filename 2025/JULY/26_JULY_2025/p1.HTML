Event Bubbling, Capturing & Delegation

Event Flow in the DOM

When an event (like click) happens on an element:

Capturing Phase – Event goes down from document → body → element.

    Target Phase – Event reaches the target element.
    
    Bubbling Phase – Event bubbles up from element → body → document.

Default Phase: Bubbling

Output will be ==>

    Child clicked
    Parent clicked  ✅ (event bubbled up)
