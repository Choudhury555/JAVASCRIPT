Common Mouse Events



    click==>Fires on single click
    dblclick==>Fires on double click
    mousedown==>When mouse button is pressed
    mouseup==>When mouse button is released
    mouseenter==>When cursor enters element (no bubble)
    mouseover==>When cursor is over the element (bubble)
    mouseleave==>When cursor leaves
    mousemove==>Fires while moving mouse over element
    contextmenu==>Right-click (context menu)
